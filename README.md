# 9-bit CPU Design Project

## Architecture
- Accumulator-based CPU
- 9-bit fixed-width instructions
- 8-bit datapath

## Instruction Format
[ opcode:4 ][ operand:5 ]

## Main Components
- ALU
- ACC register
- PC/fetch unit
- decoder
- instruction memory
- data memory

## Programs Supported
1. Hamming distance
2. 2-term multiplication
3. 3-term multiplication
